# Project Setup:
>1. RunCmd:
    `open ~/.secrets`
>2. Update the Secrets File with Token and Save - (Note: ORG_APIKEY_CHATGPT is Optional):
    `export APIKEY_CHATGPT=<Your ChatGPT API Token>`
    `export ORG_APIKEY_CHATGPT=<Organistaion API Token>`
>3. RunCmd:
    `source setup.sh`
