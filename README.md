# pyhelper
Python Library Containing Generic Utilities and Helper Functions


# Installation
> pip install pyhelper-misc


# Uses
```
# Example:
>>> from pyhelper.misc import find_all_in_dict
>>> data = {"a": {"b": "c", "d": "e"}, "f": {"g": "h"}}
>>> find_all_in_dict(data,  "d")
['e']
```
